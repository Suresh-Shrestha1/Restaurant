# Maharaja Restaturant - Restaurant Food Ordering System

A complete restaurant food ordering website built with PHP, MySQL, HTML, CSS, and JavaScript.

## Features

### Customer Features

- Browse menu with category filtering and search
- Add items to cart and manage quantities
- User registration and login with password hashing
- Secure checkout process
- Order tracking for registered users
- Fully responsive design

### Admin Features

- Secure admin login
- Dashboard with sales statistics
- CRUD operations for categories and products
- Image upload for products
- Order management with status updates
- Responsive admin panel

### Security Features

- CSRF protection on all forms
- Password hashing with bcrypt
- SQL injection protection with prepared statements
- XSS protection with output escaping
- Secure file uploads with validation
